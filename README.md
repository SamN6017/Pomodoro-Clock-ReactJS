# Pomodoro-Clock-ReactJS

This is a Pomodoro timer application built using React.

## Features
- Pomodoro timer
- Short and long breaks
- Dynamic background changes based on mode

## Technologies Used
- React: [React Documentation](https://reactjs.org/)
- CSS for styling
